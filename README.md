# Pense Bem - Ze Carioca 

Aplicativo em React Native com Expo para jogar do livro `Pense Bem - Ze Carioca na Copa do Mundo`.

## Como rodar

```bash
npm install
npm start
```

Depois, abra no Expo Go, em um emulador ou no navegador com `npm run web`.

## Estrutura principal

- `App.js`: fluxo entre tela inicial, quiz e resultado final.
- `src/data/zeCariocaProgram1.js`: arquivo com codigo de acesso, perguntas e gabarito do programa.
- `src/components/`: componentes da interface.
